# UNO Game

A Python implementation of the UNO card game using Object-Oriented Programming principles.

## About the Project

This project is a Python-based implementation of the UNO card game.
It is being developed step by step, starting with the core components of the game.

## Current Features

- Card module
  - Card representation
  - Card colors
  - Card types/values

## Technologies

- Python 3
- Object-Oriented Programming (OOP)
